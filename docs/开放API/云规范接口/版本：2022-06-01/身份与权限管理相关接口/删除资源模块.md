## 2. 输入参数


| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| AreaId | 是 | Integer | 删除资源模块的id，传入的参数必需大于0的整数 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 删除资源模块

#### 输入示例

Path : /capi/GatewayResource/DeleteResourceModule
